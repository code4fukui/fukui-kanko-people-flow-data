# 福井県観光DX AIカメラオープンデータ
[README.ja.md](README.ja.md)にある日本語のREADMEをご覧ください。

この リポジトリは、日本の福井県の主要な観光地に設置された AIカメラからの集計されたオープンデータを公開しています。データは定期的に収集、処理され、一般の使用と視覚化のために公開されています。

## デモ
[可視化アプリ](https://code4fukui.github.io/fukui-kanko-people-flow-visualization/)
[可視化アプリのソースコード](https://github.com/code4fukui/fukui-kanko-people-flow-visualization)

## 特徴
- 主要な観光地での人物、顔、ナンバープレートの検出を24時間体制でモニタリング
- データは日単位、時間単位、5分単位で集計
- 検出されたオブジェクトの年齢、性別、動きなどの詳細情報
- CSVおよびSQLiteフォーマットでデータが利用可能

## 要件
追加の要件はありません。

## 使用方法
このデータは公開されているオープンデータであり、自由にダウンロードして使用することができます。

## データ / API
福井県内の以下の場所に設置されたAIカメラから収集されたデータです:
- 福井駅東口
- 東尋坊商店街
- レインボーラインサミットパーク (駐車場1 & 2)
