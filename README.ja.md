# 福井県観光DX AIカメラオープンデータ

このリポジトリでは、福井県の主要な観光地に設置されたAIカメラから取得・集計したオープンデータを公開しています。データは定期的に収集、処理され、一般公開および可視化のために提供されています。

## 可視化

人流データを可視化するウェブアプリケーションが利用可能です。

- **[可視化アプリ](https://code4fukui.github.io/fukui-kanko-people-flow-visualization/)**
- **[可視化アプリのソースコード](https://github.com/code4fukui/fukui-kanko-people-flow-visualization)**

## データについて

### カメラ設置場所

AIカメラは、福井県の以下の主要観光地に設置されています。

- 福井駅東口
- 東尋坊商店街
- レインボーライン山頂公園（第1・第2駐車場）

### データ構造

データは集計間隔に基づいてディレクトリに分類されています。

- `full/`: 場所および検出対象ごとに分けられた、集計済みの全CSVデータ。
- `monthly/`: 日単位で集計されたデータ。`location/target/year/month` のディレクトリ構造。
- `daily/`: 時間単位で集計されたデータ。`location/target/year/month/day` のディレクトリ構造。
- `hourly/`: 5分間隔のデータ。`location/target/year/month/day/hour` のディレクトリ構造。

### データ形式

CSVファイルには、以下の3種類の検出対象に関する集計数と属性が含まれます。

- **`Person`**: 一般の人物検出。
- **`Face`**: 年齢や性別などの推定属性を含む顔検出。
- **`LicensePlate`**: 車両のナンバープレート検出と、推定される出発地（都道府県）および車両種別。

## データの利用可能性と制限

データ分析を行う際は、以下の点を考慮してください。

### 一般的な運用

- **カメラ稼働時間**: すべてのカメラは、メンテナンスや機器の故障時を除き、24時間365日稼働しています。
- **ゼロカウント**: データ内の値 `0` は、その時間帯にカメラの視野内で対象物が検出されなかったことを示します。必ずしもカメラがオフラインであったことを意味するわけではありません。

### 検出に関する注意点

- **人物検出**: 24時間の人流分析において、最も一貫性があり信頼性の高いデータを提供します。
- **顔検出**: 24時間稼働していますが、早朝（午前8時以前）および夜間（午後7時以降）の検出数は大幅に減少します。これは、暗い照明条件、実際の通行量の減少、物理的な障害物（例：福井駅の防犯シャッター）などの要因が複合的に影響しているためです。このデータは、日中（概ね午前8時から午後6時）の分析において最も信頼性が高くなります。
- **ナンバープレート検出**: カメラシステムが稼働している間は24時間利用可能です。

### 既知の問題とデータ欠損

- **データ収集開始**: データ収集は**2024年12月20日**から開始されました。この日以前のデータはありません。
- **システム停止**: **2025年9月26日〜28日**にかけて、すべての場所で完全なシステム停止が発生しました。
- **機器故障**: **レインボーライン第1駐車場**のナンバープレート検出カメラは、2026年初頭に機器の故障が発生しました。このカメラのデータは**2026年1月〜2月**の期間ほとんど利用できないため、該当期間の分析からは除外してください。

## データ処理ツール

`tools/` ディレクトリには、生データを処理・集計するためのTypeScriptスクリプトが含まれています。これらのスクリプトは [Denoランタイム](https://deno.land/) で実行することを想定しています。

- `aggregate-day.deno.ts`, `aggregate-hour.deno.ts`, `aggregate5mins.deno.ts`: 生データを日単位、時間単位、5分間隔に集計します。
- `csv2sqlite.deno.ts`: CSVデータファイルをSQLiteデータベースに変換します。
- `check-csv.deno.ts`: CSVファイルの形式と整合性を検証します。
- `license-plate-aggregation.deno.ts`: ナンバープレートデータを集計します。
- `escape-*.deno.ts`, `thin-out-*.deno.ts`: データのクリーニングと最適化を行うユーティリティスクリプト。
